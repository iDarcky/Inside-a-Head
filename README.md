# LeoLabs-Inspired Personal Site

Single-page personal website inspired by the clean and friendly direction of [leolabs.org](https://leolabs.org/), adapted into an original structure and styling.

## Files

- `index.html`: page structure and sections.
- `styles.css`: design tokens and component styles.
- `script.js`: reusable rendering logic for cards, tags, and link lists.
- `content.json`: profile, project, post, and activity content model.
- `wireframe.md`: low-fidelity layout.
- `moodboard.md`: typography, color, spacing, and component rules.

## Run Locally

Open `index.html` in a browser, or serve the folder with a simple static server.

Example using Node:

```bash
npx serve .
```

## Content Checklist

- Hero text introduces identity in one sentence.
- About section uses a short, personal tone.
- Projects are concise and include stack tags.
- Writing entries include date and read time.
- Activity section stays optional and lightweight.
